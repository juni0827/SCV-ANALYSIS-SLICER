## 🌟 핵심 강점 (Top 5)

### 1. 🤖 ML 기반 자동 분석
- **PyTorch LSTM 모델**로 최적 분석 시퀀스 자동 예측
- **50개 이상의 DSL 토큰**으로 다양한 분석 지원
- **실행 가능한 Python 코드 자동 생성**
- 업계에서 거의 유일한 접근 방식

### 2. ⚡ 뛰어난 성능
- **병렬 처리**: 멀티코어 CPU 완벽 활용
- **PyArrow 엔진**: 기본 대비 8-10배 빠른 CSV 로딩
- **캐싱 시스템**: 반복 분석 결과 재사용
- **메모리 최적화**: 30-50% 메모리 절약

### 3. 🏗️ 전문적인 아키텍처
- **모듈식 구조**: 23개 파일로 깔끔하게 분리
- **계층화 설계**: 명확한 관심사 분리
- **최근 리팩토링**: 1,972라인 → 모듈 구조
- **확장 용이**: 플러그인 추가 가능

### 4. 📊 고급 조합 분석
- **자동 컬럼 관계 탐색**: 숨은 패턴 발견
- **다양한 분석 기법**:
  - 수치형: Pearson, Spearman, Kendall 상관계수
  - 범주형: Lift, Cramér's V, Chi-square
  - 혼합형: ANOVA, Eta-squared
- **병렬 처리**: 대량 컬럼 빠르게 분석

### 5. 🎨 현대적인 UX
- **다크/라이트 테마**: 눈의 피로 감소
- **백그라운드 작업**: GUI 프리즈 없음
- **직관적인 탭 구조**: 워크플로우 기반
- **토스트 알림**: 부드러운 피드백

## 🔧 주요 기능

### GUI 애플리케이션 (app.py)
```bash
python app.py
```
- 데이터 미리보기 (페이지네이션)
- 컬럼별 통계 분석
- 7가지 시각화 차트
- 조합 분석
- 필터링 및 슬라이싱
- CSV/Excel/JSON 내보내기

### CLI 도구 (main_cli.py)
```bash
# 대화형 모드
python main_cli.py --interactive

# DSL 토큰 직접 지정
python main_cli.py --tokens C1,C2,C6 --file data.csv

# 템플릿 사용
python main_cli.py --template statistical --file data.csv
```

### DSL 시스템
- **C1-C50**: 50개 분석 토큰
- **ML 예측**: LSTM으로 최적 시퀀스 생성
- **코드 생성**: 실행 가능한 Python 스크립트

### Q2: 코딩을 모르는데 사용할 수 있나요?
**A**: 네! GUI만으로도 대부분의 분석 가능. CLI는 선택사항.

### Q3: ML 모델이 정확한가요?
**A**: DSL 예측 모델은 일반적인 분석 패턴 기반. 결과는 수정 가능.

### Q4: 상업적으로 사용 가능한가요?
**A**: 오픈소스 프로젝트. 라이센스 확인 필요 (추정: MIT).
